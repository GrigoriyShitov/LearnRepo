# Task Description

Using a for loop, the program should find the sum of the divisors of a positive integer. For example, if the input number is 12, the sum of its divisors is calculated as follows: 1 + 2 + 3 + 4 + 6 + 12 = 28.

| 2           | 56         | 120         |
| 3           | 6          | 12          |
| 4           | 78         | 168         |
| 5           | 25         | 31          |
| 6           | 1          | 1           |
